# [[Java]]
---
👉 객체 지향 언어로 개발 초창기에 이해하기 쉬운 캡슐화로 다양한 기능을 쉽게 사용 가능

### Java의 장점
1. 캡슐화가 쉽고 참조가 까다롭지 않음
2. 메모리를 직접 관리하지 않아도 돼서 쉽게 프로그램 작성 가능
### Java의 단점
1. 절차 지향 언어에 비해 컴파일 속도가 느림
2. 같은 기능에 대해 프로그램의 속도가 느림
👉 고성능 PC, 스마트폰이 출현하면서 속도 차이가 거의 느껴지지 않음
