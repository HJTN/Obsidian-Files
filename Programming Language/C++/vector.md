## Library 선언
---
``` C++
#include <vector>
```

## Vector 변수 선언
---
``` C++
vector<{Type}> {변수명};
vector<{Type}> {변수명}({Vector 크기});
```

## Vector Methods
---
### Value 추가

``` C++
vector.push({value});
```

### Value 삭제

``` C++
vector.pop();
```

### Vector의 맨 위의 Value 확인

``` C++
vector.top();
```
